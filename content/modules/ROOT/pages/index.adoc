== 概要

このワークショップでは、開発者の視点から OpenShift を実践的に紹介します。実践的でインタラクティブな演習を通して、OpenShift プラットフォーム上でコンテナ化されたアプリケーションを構築、デプロイ、管理する方法を習得します。

== 学習内容

*OpenShift 入門:*

* OpenShift コマンドラインクライアントと Web コンソールを使用する
* OpenShift 開発者インターフェースを操作し、主要な概念を理解する

*アプリケーションのデプロイと管理:*

* 既存のコンテナイメージを使用してアプリケーションをデプロイする
* Git リポジトリ内のソースコードから直接アプリケーションをデプロイする
* アプリケーションラベルを使用してコンポーネントを識別および整理する
* 変化するトラフィック負荷に対応できるようにアプリケーションをスケーリングする

*ネットワークとアクセス:*

* ルートを介してアプリケーションを外部ユーザーに公開する
* アプリケーションネットワークとサービスディスカバリを構成する

*監視とトラブルシューティング:*

* デバッグのためにアプリケーションログを表示および分析する
* 直接操作とトラブルシューティングのためにアプリケーションコンテナにアクセスする

*データベース統合:*

* OpenShift サービスカタログからデータベースをデプロイする
* アプリケーションがデータベースに接続して使用するように構成する

*コラボレーションと自動化:*

* 共同開発のために他のユーザーにアクセス権を付与する
* コードからのアプリケーションの自動ビルドを可能にするために Webhook を設定する変更点

NOTE: このワークショップではJavaアプリケーションを使用しますが、ここで説明する概念とテクニックは、.NET、Node.js、Python、その他のランタイムで構築されたアプリケーションにも適用されます。
