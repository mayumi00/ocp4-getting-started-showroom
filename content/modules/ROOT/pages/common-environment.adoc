= OpenShift アーキテクチャ
:navtitle: OpenShift アーキテクチャ

include::vars.adoc[]

image::common-environment-ocp-architecture.png[OpenShift アーキテクチャ]

このワークショップは、パブリッククラウド上で稼働する OpenShift 環境で稼働しています。この環境は、以下のシステムで構成されています。

* コントロールプレーンノード
* ワーカーノード（または「アプリケーション」ノード）
* 動的プロビジョニングストレージ
* プライベートコンテナレジストリ
* ルーティングレイヤー
